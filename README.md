# 開発言語
* Ruby3.0.1
* Rails6.0.3

# 実行手順
```
$ git clone git@github.com:yuki-3110/moving_support.git
$ cd moving_support
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
